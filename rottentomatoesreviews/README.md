# Rotten Tomatoes Movie & TV Reviews Analysis

This project analyses over 3,000 movies and TV shows to explore patterns in critic and audience ratings over time. The analysis examines how different genres are received by critics versus audiences and investigates actor popularity based on aggregated review data. The goal is to uncover trends in audience preferences and critical reception using structured, reproducible data analysis.

The dataset is not included in this repository due to Kaggle licensing.
